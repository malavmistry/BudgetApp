BudgetApp Docker startup

1. Open PowerShell in the repository root:
   D:\repos\BudgetApp-master\BudgetApp-master

2. Build and start the application and SQL Server containers:
   docker compose up --build -d

3. Check that both containers are healthy:
   docker compose ps

4. Open the application in a browser:
   http://localhost:8080

5. To stop the stack:
   docker compose down

Notes:
- The SQL Server password is read from the .env file in this folder.
- The app container connects to SQL Server by the Docker service name 'sqlserver', not localhost.
- On first startup, the app applies Entity Framework migrations automatically.
- If you change the SQL password, update SA_PASSWORD in .env before running docker compose up --build -d.
